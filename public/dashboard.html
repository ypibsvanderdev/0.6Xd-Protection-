API Reference

        

          

            
GET /v1/validate/:scriptId/:key

            
Validate a key. Returns VALID or error message.

            
GET https://api.06xdprotect.com/v1/validate/sc_abc123/KEY-XXXX-XXXX

Response: VALID

          

          

            
GET /v1/load/:scriptId/:key

            
Returns the protected script source if key is valid.

            
GET https://api.06xdprotect.com/v1/load/sc_abc123/KEY-XXXX-XXXX

Response: [protected Lua source]

          

          

            
POST /v1/keys/generate

            
Generate a new key for a script.

            
POST https://api.06xdprotect.com/v1/keys/generate
Authorization: Bearer sk-06xd-...
{
  "scriptId": "sc_abc123",
  "expiresIn": 86400,
  "note": "user123"
}

Response: { "key": "KEY-XXXX-XXXX-XXXX", "expires": "2025-12-31" }
