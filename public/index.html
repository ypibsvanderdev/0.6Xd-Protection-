Protection Layers

Built different.
Protected differently.

Every script goes through 6 layers of transformation before leaving our servers.

🧬
String Table Encoding
All string literals extracted, XOR-encoded with a random key per script, stored in an encrypted table only resolved at runtime.
🌀
VM Dispatch Layer
Your code is compiled into a custom bytecode and executed through a Lua virtual machine. Decompilers see only the VM, never your logic.
🔑
Key Validation
Scripts phone home to our API on every execute. Invalid, expired, or revoked keys are rejected within milliseconds.
🛡️
Anti-Tamper
An integrity hash is baked into the loader. Any modification to the protected output causes it to silently halt.
👁️
Anti-Logger
Hooks SimpleSpy, RemoteSpy, and 15+ known logger globals. Detection kicks the executor user instantly.
♾️
Identifier Mangling
Every variable, function, and local name is replaced with lookalike Unicode glyph identifiers that are visually indistinguishable.
